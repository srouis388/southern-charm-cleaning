Southern Charm Cleaning Landing Page

A responsive landing page created for Southern Charm Cleaning, a residential cleaning company serving Douglas, Broxton, and surrounding communities in South Georgia.

This project focuses on providing a clean, modern online presence that helps potential customers quickly learn about the business and request a free estimate.

⸻

Features

* Responsive design for desktop, tablet, and mobile devices
* Modern hero section
* Service overview
* Call-to-action buttons
* Embedded Google Forms estimate request
* Contact information
* SEO-friendly HTML structure
* Hosted with Cloudflare Pages

⸻

Services Highlighted

* Residential Cleaning
* Move-In / Move-Out Cleaning
* Deep Cleaning
* Restoration Cleaning

⸻

Technology

* HTML5
* CSS3
* GitHub
* Cloudflare Pages
* Google Forms

⸻

Purpose

The goal of this project is to establish a professional online presence for a small local business while keeping the website simple, fast, and easy to maintain.
